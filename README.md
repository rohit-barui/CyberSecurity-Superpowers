# CyberSecurity Superpowers

A cybersecurity extension for [Superpowers](https://github.com/rohit-barui/superpowers) by **rohit-barui** (MIT license).

Adds five security-focused skills to the Superpowers agent workflow:
- **threat-modeling** — STRIDE/CVSS/MITRE ATT&CK threat analysis
- **secure-coding** — language-specific OWASP checklists
- **static-analysis** — SAST & dependency vulnerability scanning
- **penetration-testing** — red-team plan generation
- **incident-response** — NIST 800-61 aligned playbooks

# Incident Communication Templates

## Internal Alert
```
Subject: [SEVERITY] Security Incident - [Brief Description]

Time Detected: [Timestamp]
Affected Systems: [List]
Current Status: [Investigation/Containment/Recovery/Resolved]
Point of Contact: [Name]

Details:
[Brief description of incident and actions taken]
```

## Customer Notification (if needed)
```
Subject: Security Notice - [Date]

We are writing to inform you of a security incident that may affect your account.
We have taken immediate steps to contain and investigate the issue.

What happened: [Description]
What we are doing: [Actions]
What you should do: [Customer actions if any]
```

```
Subject: Incident Resolution - [ID]

The security incident has been resolved.

Actions taken: [List]

# C4 Model Guide

## Abstraction Levels

1. **Context** - System scope and external dependencies
2. **Container** - High-level technology choices
3. **Component** - Internal decomposition of containers
4. **Code** - Class-level diagrams (optional)

## Security Extensions

- Add trust boundaries at Context level
- Add security controls at Container level
- Add threat references at Component level
